Описание

Проект представляет собой сервис для студентов, в котором при регистрации автоматически загружается расписание для выбранной группы (путем парсинга страниц вузов). У старосты группы есть возможность добавлять учебные задания. Удобства придает интерактивный календарь, в котором можно увидеть текущие задачи группы.
